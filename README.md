# Finance-Working-Files

Welcome to the **Finance-Working-Files** repository! This collection is dedicated to finance and investment-related files aimed at analysis and modeling.

## Repository Structure

The repository is organized into the following directories:

- **DCF_Code**: Contains some textbook code implementations related to Discounted Cash Flow (DCF) analysis.
- **DCF_templates_excel**: Includes Excel templates for conducting DCF evaluations.
- **Investment_2025**: Contains textbook examples of mean-variance portfolio optimisation, elementary intraday trading framework on the Nasdaq index, some further analysis. 
- **Portfolio_Optimisation**: Offers resources and code for optimizing investment portfolios.
- **Time_Series_Analysis**: Experimenting multiple techniques and models of Financial Econometrics.
